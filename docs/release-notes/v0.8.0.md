# v0.8.0

## Fix Token and Cost Tracking Inflation

- Fixed a bug where session token counts and cost were massively inflated over time
- Claude Code reports cumulative totals in each result message, but AgentDock was summing them as incremental deltas — causing quadratic growth over many interactions
- Session stats now correctly replace cumulative values and compute per-interaction deltas for display

## Sound Notifications

- Plays Windows **device connect** sound when a session starts
- Plays Windows **message nudge** sound when an agent finishes and is waiting for user input (diamond turns green)
- Plays Windows **device disconnect** sound when a session ends
- Uses the user's configured Windows sound scheme — respects mute and custom sounds

## Shared Markdown Rendering

- Extracted shared MarkdownHelper so AI chat and file preview use the same rendering pipeline
- AI chat now gets the same bold/italic link fixups and code block theming as file preview

## Slash Command Autocomplete

- Type `/` in the chat input to see a dropdown of all available Claude Code slash commands with descriptions
- Up/Down arrows to navigate, Enter or Tab to insert, Escape to dismiss
- Filters as you type — `/com` narrows to `/compact` etc.
- Includes Claude Code 2.1.x commands from the official docs plus Agent Dock local commands (`/clear`, `/stop`, `/logs`)
- Primary command names only; aliases (e.g. `/settings` → `/config`) are shown in the description

## Model Name in AI Chat Panel Title

- AI chat panel title now shows the active Claude model (e.g. `Sonnet 4.5 — $0.0123 · 1.2k tokens`) once the session reports it
- Raw model ids like `claude-sonnet-4-5-20250929` are shortened to `Sonnet 4.5` (family + version, trailing date dropped)
- Falls back to `AI Chat` before the session starts

## Plan Usage Indicator in Title Bar

- Shows the current 5-hour session utilization and time-to-reset in the title bar (e.g. `5h: 24% · resets in 2h 14m`)
- Click for a popup with the full breakdown: 5-hour window, 7-day weekly window, per-model (Opus/Sonnet) usage where available, and extra-usage credits
- Refreshes automatically every 60 seconds; manual refresh button in the popup
- Data sourced from the same `GET /api/oauth/usage` endpoint Claude Code's `/status` tab uses (no inference tokens consumed)
- Shows sign-in / offline / expired states when the OAuth token from `~/.claude/.credentials.json` isn't usable
